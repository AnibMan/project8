1.git init >>>creates new repo in your directory
2.git status >>> shows status/tracking of uncommitted files
3.git add . OR, git add -A  >>> add all the file to staging for commit. If the file is again changed then the changes will not be commited when commit is done. So we again have to add once more to commit all updated changes.
4.git commit -m "message" >>> commit the git
5.git commit -a -m "change shortcut" >>> this is shortcut to add


Clone your project
Go to the folder application using cd
Run composer install on your cmd or terminal
Copy .env.example file to .env on root folder. You can type copy .env.example .env if using command prompt Windows or cp .env.example .env if using terminal Ubuntu
Open your .env file and change the database name (DB_DATABASE) to whatever you have, username (DB_USERNAME) and password (DB_PASSWORD) field correspond to your configuration. 
By default, username is root and you can leave password field empty. (This is for Xampp) 
By default, username is root and password is also root. (This is for Lamp)
Run php artisan key:generate
Run php artisan serve
Go to localhost:8000
